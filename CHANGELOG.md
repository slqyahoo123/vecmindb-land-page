# VecminDB 迭代历史 (Changelog)

## [v1.5.0] - 2024-04-15
### 🛡️ 品牌与叙事重大进化 (AI Agent Memory Edition)
- **定位对齐**：全量重塑项目叙事，从“向量数据库”晋升为“AI Agent 长效记忆与动态知识中枢”。
- **商业资产包**：新增专业级 Cold Outbound 脚本、Pro 版销售手册、外呼 SOP 及商业 BP 大纲。
- **技术背书**：新增 Mermaid 架构蓝图，详细拆解 NSGA-II 算法与 1024 路 Manifold 锁的技术逻辑。

### 🎨 交互与性能优化
- **UI 对齐**：全站文案（中英双语）完成 Agent 叙事对齐。
- **移动端优化**：修复了手机端导航链接点击后不自动关闭菜单的交互缺陷。
- **安全性增强**：完成了 Nginx 安全头与 W3C 语言属性同步的最佳实践部署。

### 🛠️ 工具链与合规性
- **实战指南**：新增真正的 Waitlist 后端集成对接指南。
- **合规件**：落地了 1.0 版《隐私政策》与《服务条款》框架。
- **自动化**：重构了资产看板生成器，覆盖全量商用件。
