# VecminDB: 为 10B 阶 AI Agent 打造的长效数字化记忆
**上海灵心智算智能科技有限公司 官方品牌资产包 (v2.0)**

---

## 🌩️ 项目愿景 (Vision)
VecminDB 旨在重新定义 AI 时代的“智算感知底座”。我们利用 NSGA-II 自进化引擎与 1024 路 Manifold 分片锁机制，为全球开发者提供工业级的 AI Agent 长效记忆与动态知识管理方案。

## 🛠️ 核心资产概览 (Asset Map)

### 1. 数字化门户 (Digital Portal)
- **极智视觉前端**: `index.html` (3D 向量动效 / i18n 双语 / 响应式).
- **多端美学定义**: `style.css` (Aurora 流光效果 / 玻璃拟态设计).
- **交互与分析引擎**: `script.js` (集成 W3C 合规性 lang 切换与 Telemetry 追踪).

### 2. 商业作战武器 (Business Arsenal)
- **销售脚本 SOP**: `docs/sales/` (含冷启动脚本、博弈手册及外呼 SOP).
- **商业 PPT 大纲**: `BUSINESS_DECK_OUTLINE.md` (12 页核心展示逻辑).
- **架构蓝图**: `docs/infrastructure/Architecture_Overview.md` (Mermaid 技术背书).

### 3. 工程与运维 (Engineering & Ops)
- **生产容器化**: `Dockerfile`, `nginx.conf` (硬核安全加固).
- **自动化看板**: `scripts/gen_dashboard.js` (生成 `ASSET_DASHBOARD.html`).
- **数据仿真器**: `scripts/demo_emulator.py` (模拟 10B 级向量空间脉冲).

## 🚀 极速启动 (Quick Start)
1. **环境初始化**: `bash scripts/setup.sh`
2. **启动本地预览**: `npm run dev`
3. **资产全案检阅**: 点击根目录下的 `ASSET_DASHBOARD.html`
